# אקדמיית הסוכנים — אתר קורס סטטי

זהו אב־טיפוס ראשון לאתר קורס בעברית ללמידת בניית סוכני AI מעל כמה LLMs.

## איך פותחים
1. חלץ את קובץ ה־ZIP.
2. פתח את `index.html` בדפדפן.
3. אין צורך בשרת או התקנות.

## מה יש באתר
- דף בית
- מסלול למידה
- זירת LLM: OpenAI, Claude, Gemini ומודלים פתוחים
- מעבדת סוכנים עם טופס Agent Spec
- מילון מונחים לאנשים שאינם מהנדסי תוכנה
- תרגילים ותבניות
- פרויקט מסכם: ראש לשכה דיגיטלי

## מה השלב הבא
- להוסיף תוכן שיעורים מלא.
- להוסיף דוגמאות וידאו/שקפים.
- לחבר התקדמות ל־Google Sheets.
- להפוך את מעבדת הסוכנים למחולל Prompts/JSON אמיתי.
