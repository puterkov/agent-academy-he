זהו אתר קורס מעשי בעברית, למי שאינו מהנדס תוכנה אבל רוצה להפוך למקצוען בבניית סוכנים: להבין מושגים, להשוות LLMs, לתכנן כלים, להגדיר הרשאות, לבנות מעבדת ניסוי ולסיים עם ראש לשכה דיגיטלי.

עדכון האתר בגיטהאב

איך מעלים את גרסה 2 לאתר שכבר פרסמת

  1. חלץ את קובץ ה־ZIP החדש.
  2. פתח את ה־repository שלך ב־GitHub: agent-academy-he.
  3. לחץ Add file → Upload files.
  4. גרור את הקבצים החדשים: index.html, styles.css, data.js, app.js, README.md.
  5. אשר החלפה של הקבצים הישנים אם GitHub שואל.
  6. למטה לחץ Commit changes.
  7. המתן דקה־שתיים ורענן את האתר.

אם האתר לא מתעדכן מיד, פתח אותו בחלון פרטי או לחץ Ctrl+F5.
